# Vision-Crafters

## AI-Powered Candidate Ranking System

Vision-Crafters is an AI-powered hiring assistant that ranks candidates based on job descriptions using semantic analysis and intelligent scoring.

## Features
- Resume Parsing
- Job Description Analysis
- Candidate Ranking
- AI-Based Matching
- Recruiter-Friendly Dashboard
